

# Редизайн анимации открытия колоды карт

## Концепция

Вместо текущего горизонтального скролла (как CS:GO кейсы) — сделать **вертикальную рулетку** в стиле первого скриншота: карточки прокручиваются сверху вниз (или снизу вверх) по центру экрана, замедляясь и останавливаясь на выигрышной. Вокруг — эффекты свечения в стилистике игры (пурпурный/магический).

## Дизайн

```text
┌──────────────────────────────────────┐
│          🎴 ОТКРЫТИЕ КОЛОДЫ          │  Заголовок
│                                      │
│    ┌─────────────────────────┐       │
│    │  ╔═══╗  ╔═══╗  ╔═══╗   │       │  Верхний ряд (размытый,
│    │  ║   ║  ║   ║  ║   ║   │       │  полупрозрачный)
│    │  ╚═══╝  ╚═══╝  ╚═══╝   │       │
│    │─────── ИНДИКАТОР ───────│       │  ← Линия выбора
│    │  ╔═══╗  ╔═══╗  ╔═══╗   │       │  Центральный ряд
│    │  ║ ★ ║  ║ ★ ║  ║ ★ ║   │       │  (яркий, подсвеченный)
│    │  ╚═══╝  ╚═══╝  ╚═══╝   │       │
│    │  ╔═══╗  ╔═══╗  ╔═══╗   │       │  Нижний ряд (размытый)
│    │  ║   ║  ║   ║  ║   ║   │       │
│    │  ╚═══╝  ╚═══╝  ╚═══╝   │       │
│    └─────────────────────────┘       │
│                                      │
│    [Пропустить]  [Пропустить все]    │
│    ═══════════ прогресс ═══════════  │
└──────────────────────────────────────┘
```

**Уточнение**: Горизонтальная лента (как сейчас), но с улучшениями:
- Карточки за пределами центра — уменьшены (scale 0.8) и затемнены (opacity 0.5)
- Центральная зона подсвечена пурпурным свечением
- Индикатор — два треугольника сверху и снизу (как стрелки)
- При остановке — выигрышная карта увеличивается, появляются частицы/искры
- Фон с анимированным градиентом вместо простого `bg-black/90`

## Изменения

### Файл: `src/components/game/dialogs/CardPackAnimation.tsx`

1. **Анимированный фон** — радиальный градиент с пульсацией в цветах game-primary/game-accent
2. **Улучшенные карточки в ленте**:
   - Больший размер (w-36 h-56 вместо w-32 h-52)
   - Карточки вне центра: scale(0.85), opacity 0.4, brightness(0.5)
   - Центральная карточка: scale(1.05), полная яркость
   - Градиентные маски слева/справа (fade-to-transparent)
3. **Индикатор** — заменить красную линию на два пурпурных треугольника (сверху и снизу), с пульсирующим свечением
4. **Эффект при остановке**:
   - Выигрышная карта: scale 1.2, золотая рамка, glow-эффект
   - Текст "Ваша добыча!" с анимацией появления
   - Мерцающие частицы (CSS pseudo-elements)
5. **Сокращение длительности**: 10сек → 6сек (быстрее, динамичнее)
6. **Замедление**: easing `[0, 0.7, 1]` → более плавное торможение в конце
7. **Кнопки**: стилизация под game-primary/game-accent с иконками

### Не меняется
- `CardRevealModal.tsx` — логика показа результата остается
- `useCardPackOpening.ts` — хук без изменений
- Props интерфейс `CardPackAnimationProps` — без изменений
